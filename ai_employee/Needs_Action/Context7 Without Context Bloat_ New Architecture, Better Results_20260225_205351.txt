What Changed

Previously, LLMs would search and filter documentation repeatedly, consuming tokens with each round trip. Now, Context7 performs high-quality reranking on our servers using fast reranking models before returning results.

This means:

 * More relevant documentation in fewer tokens
 * Faster responses with less back-and-forth
 * Consistent, predictable results
